Use Case Descriptions:

Use Case 1: Register User

The Actor in this Use Case is a New Customer. This use Case allows the New Customer to be able to register as a user and become a Registered Customer. For this use case the New Customer will Access the Website which will take them to the Home Page. From the Home Page they will click on Become a Customer which will bring them to the Sign in Page. Then the New Customer will enter the required information to become a Registered Customer and the system will return this as a logged in User

Use Case 2: Customer Login

The Actor in this Use Case will be a Registered Customer. This Use case allows a Registered Customer to log into their account. The Registered Customer will go onto the website which will bring them to the HomePage. Then the Regustered Customer will click the Login Button which will bring thme to the Login Page. Then the Registered Customer will enter the required information which will log them into their account

Use Case 3: Bid With tokens (includes use cases View Countdowns, View Bid History, View Title & Description, and Access Active Auctions)

The Actor in this Use Case will be a Registered Customer. This use case will allow a Registered Customer to bid on an auction item item with tokens. The Registered Customer will go to the Website where they are on the HomePage Loggedin. Then they will click on live/active auctions where they will see the title and description for each auction, access all active auctions, and view the countdown. Next, the Registered Customer will click on a specific auction item will bring them to the item page which will allow them to view the bid history of the item. Lastly, the Registered Customer will be able to bid on that item with their tokens which the system willreturn with "x Item Successfully Bid On".

Use Case 4: Add Auction Item

The Actor in this Use Case will be a Registered Customer. This use case will allow a Registered Customer to Add an Auction Item. The Registered Customer will go onto the website where they are logged in. Then, they will access the auction page. Next, the Registered Customer will calick the button add auction item which will bring them to the add item page. Then, the Registered Customer will enter the required item information then the system will return with "Item Successfully added".
